## 🚨 The Big News

Anthropic's Claude Design Launches with AI Visual Collaboration

Anthropic dropped Claude Design today, marking their first major push into visual creation tools. This new Labs product turns Claude into a collaborative design partner that can create everything from polished slides to full prototypes.

**What's new:**
- Real-time collaboration for creating designs, prototypes, slides, and one-pagers
- Visual work generation that goes beyond text responses
- Integration with existing Claude workflows and capabilities
- Part of Anthropic Labs experimental product lineup

**Why this matters:** Most AI tools still live in text-only worlds, but actual work involves plenty of visual communication. Claude Design could bridge that gap for teams building presentations, mockups, or visual documentation without needing separate design tools.

## Claude Code

What's new from Anthropic's AI coding assistant

🔄
**Improved · Performance**
**Native binary replaces JavaScript bundling**
Claude Code v2.1.113 swapped out its bundled JavaScript approach for native binaries that run per-platform. Should feel snappier and more responsive during heavy coding sessions.

[Source →](https://github.com/anthropics/claude-code/releases/tag/v2.1.113)

🛠️
**New · Security Controls**
**Network sandbox gets domain deny-listing**
You can now block specific domains even when broader wildcards would allow them through the `sandbox.network.deniedDomains` setting. Gives you more granular control over what your coding environment can access.

[Source →](https://github.com/anthropics/claude-code/releases/tag/v2.1.113)

🔧
**Fixed · UI Polish** 
**Better fullscreen editing and Remote Control**
Shift+arrow keys now scroll properly when selecting past screen edges in fullscreen mode. Remote Control clients can finally query @-file autocomplete suggestions, and long URLs stay clickable when they wrap.

[Source →](https://github.com/anthropics/claude-code/releases/tag/v2.1.113)
