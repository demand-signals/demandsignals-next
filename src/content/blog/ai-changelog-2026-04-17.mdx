## 🚨 The Big News
Anthropic's Claude Design Launches

Today, Anthropic launched Claude Design by Anthropic Labs — a new product that lets you collaborate with Claude to create polished visual work like designs, prototypes, slides, one-pagers, and more.

**What's new:**
- Visual collaboration tool that works directly with Claude to create designs
- Supports multiple types of visual content including presentations, prototypes, and documents
- Part of Anthropic Labs, indicating it's an experimental/preview product
- Available as of April 17, 2026

**Why this matters:** This marks Anthropic's expansion beyond text-based AI into visual creation tools, potentially competing with design platforms and making professional design work more accessible to non-designers.

## Claude Code
What's new from Anthropic's AI coding assistant

🛠️
**New · Native Performance**
**Claude Code now runs faster with native binaries**
The CLI has been upgraded to use native platform-specific binaries instead of bundled JavaScript, which should improve performance and startup times.

🔧
**Fixed · Multiple Bugs**
**Better stability across the board**
Fixed issues with copying text character counts, Windows file insights crashes, prompt cursor visibility, and several Remote Control session problems.

⚠️
**Improved · Security**
**Enhanced sandbox protection on macOS**
Added better protections for sensitive system directories like `/private/{etc,var,tmp,home}` and improved command filtering for wrapped commands using `env`, `sudo`, and similar tools.

🧠
**New · Network Controls**
**Block specific domains even with wildcards**
Added a new `sandbox.network.deniedDomains` setting that lets you block specific domains even when broader wildcard rules would normally allow them.
