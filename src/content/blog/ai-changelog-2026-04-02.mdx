## Claude Code
What's new from Anthropic's AI coding assistant

🛠️
**New · MCP Tools**
**MCP tools can now handle way bigger results without being cut off**
Previously, MCP tool results got truncated at a smaller limit. Now developers can tag specific results to pass through up to 500K characters — perfect for things like database schemas or large API responses that Claude needs to see in full.

⚡
**Improved · Performance**
**The terminal got faster and smarter across multiple areas**
The tool now strips formatting codes faster (especially on Bun), uses shorter text anchors to save tokens, and loads conversations more reliably when you resume sessions. Plus several terminal emulators now properly handle the cmd+delete shortcut.

🔧
**Fixed · Bugs**
**Plan mode works better in remote sessions · Conversation history stops disappearing · JSON validation catches bad settings**
The biggest fix prevents losing your chat history when transcript files fail to save silently. Plan mode also remembers your plan file after container restarts, and the settings validator now properly catches permission configuration errors.

🛠️
**New · Plugin System**
**Plugins can now include their own executable files**
Plugin developers can ship executables in a `bin/` folder and call them directly from Claude's Bash tool. This opens up more possibilities for custom tooling and integrations.

🔑
**New · Security Controls**
**Added option to disable shell execution in skills and plugins**
You can now turn off inline shell execution across skills, custom slash commands, and plugin commands with the new `disableSkillShellExecution` setting. Useful if you want Claude's help but prefer tighter security controls.
