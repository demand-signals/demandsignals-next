## Claude Code
What's new from Anthropic's AI coding assistant

🛠️
**New · Worktree management**
**You can now jump between different code branches using the path parameter**
This makes it way easier to work on multiple features at once or compare different versions of your code. Think of it like having multiple workspaces that Claude can switch between instantly.

⚡
**New · Proactive mode alias**
**Typing /proactive now does the same thing as /loop**
Just a shortcut that makes more sense when you want Claude to keep working autonomously. Both commands still do the exact same thing, just different ways to say it.

🔄
**Improved · Network handling**
**API streams won't hang forever anymore - they timeout after 5 minutes**
If your internet connection gets wonky or the API stops responding, Claude Code will now give up after 5 minutes and try a different approach instead of sitting there spinning forever.

🔧
**Fixed · Bugs**
**Images in queued messages stay attached · Screen doesn't go blank on long prompts · ASCII art keeps its formatting · Keyboard shortcuts work properly**
Bunch of annoying little issues that were breaking the flow. The ASCII art fix is especially nice if you're working with diagrams or formatted text output.
