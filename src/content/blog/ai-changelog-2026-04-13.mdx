## Claude Code
What's new from Anthropic's AI coding assistant

🧠  
**New · Worktree Tools**
**Claude can now jump between existing Git worktrees in your repository**
You can tell Claude to switch into a specific worktree you've already set up using the new path parameter. This makes it much easier to work with multiple branches or feature work at once.

🛠️  
**New · Plugin Monitors**
**Background monitoring for plugins that auto-start when you begin a session**
If you've built plugins that need to watch for certain conditions, they can now run background monitors automatically. These kick in right when your session starts or when you invoke a skill.

🔄  
**Improved · Loop Commands**
**`/proactive` is now just another way to say `/loop`**
Both commands do the same thing now, so use whichever feels more natural. This command lets Claude work continuously on tasks without waiting for you to prompt it each time.

🔧  
**Fixed · Bugs**
**Better handling of network timeouts · Fixed image attachments getting lost · No more blank screens when typing long prompts**
Several annoying issues are now resolved: Claude won't hang forever on slow API connections, images you attach while Claude is working won't disappear, and your screen won't go blank when your message wraps to multiple lines.

⚠️  
**Improved · Network Handling**
**Clearer error messages and automatic retries when connections fail**
Instead of showing a silent spinner when something goes wrong with the connection, you'll now get helpful retry messages immediately. Claude also gives up on stalled streams after 5 minutes instead of waiting forever.
