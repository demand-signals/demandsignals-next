# The AI ChangeLog - Sunday, April 5, 2026

Saturday was quiet across most platforms, but Claude Code had its busiest day in weeks with a major release packed with new features and fixes.

## Claude Code
What's new from Anthropic's AI coding assistant

🔑
**New · Security**
**Your IT team can now force-refresh security settings on startup**
Added a new policy that makes Claude Code wait for the latest security settings before starting up, and shuts down if it can't get them. This one's for companies that need fail-safe security controls.

🛠️
**New · Setup**  
**AWS Bedrock setup now walks you through everything step by step**
No more wrestling with AWS credentials and regions. When you pick "3rd-party platform" on the login screen, you get a friendly wizard that handles authentication, picks your region, tests everything, and sets up your model preferences.

📊
**New · Cost tracking**
**Pro users get detailed breakdowns of exactly what's costing them money**
The `/cost` command now shows per-model spending and how much you're saving from prompt caching. Finally know which models are eating your budget.

⚡
**New · Version picker**
**Release notes are now interactive — pick any version to compare**
Type `/release-notes` and you can browse through different versions instead of just seeing the latest. Handy when you're trying to figure out when something changed.

🤖
**New · Remote sessions**
**Your computer name shows up in remote session names automatically**
Instead of random names like "graceful-unicorn", you'll see "myhost-graceful-unicorn" so you can tell which machine is which. You can customize the prefix if you want.

🔄
**Improved · Caching**
**Pro users get a heads up when their cache expires**
When you come back to a session and the cache has expired, you'll see roughly how many tokens are about to be sent fresh. No more surprise billing spikes.

🔧
**Fixed · Bugs**
**Major fixes for tmux crashes, API errors, and survey spam**
Fixed subagents breaking after tmux windows get killed · Stop hooks failing incorrectly · API 400 errors from extended thinking · Accidental survey submissions from keypresses · Misleading interrupt hints · Homebrew update channel confusion · Editor cursor jumping issues · Message duplication in some terminals · Wrong token counts in hints · MCP server connection problems.
