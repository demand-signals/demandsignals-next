## Claude Code
What's new from Anthropic's AI coding assistant

🔧
**Fixed · Bugs**
**Multiple fixes keep your coding sessions smooth**
Fixed a bunch of annoying issues: tmux sessions won't break when windows get renumbered, tool validation won't crash on weird JSON, and extended thinking won't throw random API errors. Plus no more accidental feedback submissions when you're rapidly typing.

⚡
**Improved · Speed**
**File diffs now process 60% faster on large files**
If you're working with big files that have lots of tabs or special characters, you'll notice the Write tool is way snappier. This is one of those behind-the-scenes improvements that just makes everything feel more responsive.

🔑
**New · Authentication**
**Interactive Bedrock setup wizard guides you through AWS connection**
No more fumbling around with AWS credentials and regions. When you select "3rd-party platform" from the login screen, a new wizard walks you through connecting to AWS Bedrock step by step—authentication, regions, the works.

🛠️
**New · Enterprise controls**
**New policy forces fresh remote settings on startup**
This one's for enterprise admins. The `forceRemoteSettingsRefresh` setting makes Claude Code wait for the latest managed settings before starting up, and it'll fail completely if it can't fetch them. Basically ensures everyone's running with the current company policies.

📊
**Improved · Cost tracking**
**Cost breakdown now shows per-model and cache hits for subscription users**
The `/cost` command got smarter—now you can see exactly which models you're using and how often your prompt cache is saving you money. Great for understanding where your usage is going.

🔄
**Improved · Session management**
**Remote Control sessions now use your computer name as default**
Instead of random names, your remote sessions will default to something like `myhost-graceful-unicorn` (you can still change the prefix). Makes it way easier to tell which session belongs to which machine.
