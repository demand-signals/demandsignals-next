**New · Display modes**
**Run `/tui fullscreen` to get flicker-free rendering in the same conversation**
If you've been dealing with screen flickering while coding, this new command switches Claude Code to a smooth fullscreen interface. Think of it like switching from windowed to fullscreen mode in a video game - same conversation, just smoother visuals.

⚡
**New · Focus controls**
**Focus view now has its own `/focus` command instead of sharing with verbose mode**
Before, pressing Ctrl+O toggled through multiple view modes which was confusing. Now you get dedicated controls: Ctrl+O just switches between normal and verbose transcript, while `/focus` gives you that distraction-free coding view.

🛠️

**Resume now brings back your scheduled tasks that haven't expired yet**
