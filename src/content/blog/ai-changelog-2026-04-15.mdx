## TL;DR

OpenAI rolled out their new "Agents SDK" — think AI assistants that can actually take action instead of just giving advice. This is their biggest developer platform update in months. Meanwhile, Claude Code (Anthropic's coding tool) got a major refresh with a new fullscreen mode that developers are gonna love.

## OpenAI

**What changed:**
- Released the Agents SDK in public beta — lets you build AI assistants that can actually do things (like book appointments, manage files, or control other software) instead of just chatting
- Added Agent Builder — a visual tool to create these AI assistants without coding, like a drag-and-drop interface for AI
- New ChatKit component — ready-made chat interface you can drop into your website or app
- Added "Computer Use" tool — AI can now control your computer screen, click buttons, fill forms (yes, it's as wild as it sounds)
- Voice agents now available — AI that can handle phone calls or voice conversations

**Why you should care:** If you've been waiting for AI that can actually complete tasks instead of just suggesting them, this is huge. The Agent Builder means non-technical folks can create useful AI assistants. Computer Use is honestly a bit scary but incredibly powerful for automating repetitive tasks.

## Claude Code (Anthropic)

**What changed:**
- Added fullscreen mode (`/tui fullscreen`) for distraction-free coding sessions
- New focus view to hide clutter when you're deep in code
- Better plugin management — favorites and important items show up first
- Session recap feature now works even if you have analytics turned off
- Fixed a bunch of crashes and UI glitches that were annoying developers

**Why you should care:** This one's mainly for developers — if you use Claude Code for programming, the fullscreen mode is a game-changer for staying focused. Skip if coding isn't your thing.

## The Bottom Line

If you run a business and have been curious about AI automation, OpenAI's new Agents SDK is worth exploring — especially the Agent Builder if you're not technical. The "Computer Use" feature could automate tons of repetitive computer tasks, though maybe start small and see how it goes. For developers using Claude Code, definitely try the new fullscreen mode. Everyone else can sit this one out — these are power-user features.
