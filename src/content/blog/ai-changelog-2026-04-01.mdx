## Claude Code
What's new from Anthropic's AI coding assistant

⚡
**New · Interactive learning**
**Claude Code now teaches itself to you with animated tutorials**
Hit `/powerup` to launch interactive lessons that walk you through Claude's features with actual demos. Finally, a way to learn all those keyboard shortcuts without reading documentation.

🛠️
**New · Background jobs**
**Pause mid-task and pick up where you left off across devices**
New "defer" option lets you pause Claude when it wants to run a command, then resume later with `-p --resume`. Perfect when you need to switch contexts or review something before letting it proceed.

🤖
**New · Digital pet**
**Your new coding companion has arrived (April Fools!)**
Meet `/buddy` — a tiny creature that hatches and watches you code. It's completely useless but oddly satisfying to have around.

Large conversations no longer slow down over time · Multi-project resume loads in parallel · New flicker-free rendering option with `CLAUDE_CODE_NO_FLICKER=1` · Better handling of huge files without crashes.
