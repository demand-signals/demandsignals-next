## Google Gemini
What's new from Google's AI platform

🧠
**New · Pricing**
**Google introduces Flex and Priority tiers for Gemini API**
Google rolled out two new inference tiers for the Gemini API. "Priority" gets you faster responses with guaranteed availability, while "Flex" is the budget option — slower but cheaper. If you're building apps on Gemini, this lets you pick the right balance between speed and cost.

## Claude Code
What's new from Anthropic's AI coding assistant

🧠
**New · Interactive Training**
**Learn Claude Code with hands-on lessons and animated demos**
Added a new `/powerup` command that teaches you Claude Code features through interactive lessons with animations. Think of it as a built-in tutorial system that actually shows you what each feature does instead of making you hunt through docs.

🔄
**Improved · Performance**
**Long conversations won't slow you down anymore**
Fixed several performance issues where SDK sessions and SSE transport would get slower and slower as your conversation got longer. Now your 500th message loads just as fast as your first one.

🔧
**Fixed · Bugs**
**Auto mode stops pestering you · Resume works properly · File editing is more reliable**
Fixed an infinite loop where the rate-limit dialog kept popping up and crashing your session · Fixed `--resume` losing your prompt cache on the first request · Fixed Edit/Write tools failing when auto-formatting kicked in between edits.

⚡
**New · Offline Support**
**Claude Code Marketplace works even when git fails**
Added `CLAUDE_CODE_PLUGIN_KEEP_MARKETPLACE_ON_FAILURE` environment variable so your existing plugins stay cached when `git pull` fails. Perfect for working in environments with spotty internet or restricted git access.

🎨
**New · April Fools Feature**
**Meet your new coding buddy**
Added `/buddy` — hatch a small creature that watches you code. It's here for April 1st, so enjoy your digital pet while it lasts!
