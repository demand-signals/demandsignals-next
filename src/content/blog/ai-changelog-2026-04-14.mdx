Quiet day across the board today. While Claude Code saw significant updates with new caching features and quality-of-life improvements, the major AI platforms — OpenAI, Anthropic's main API, Google Gemini, and DeepSeek — didn't release any notable changes on Tuesday, April 14, 2026.

## Claude Code
Two feature-packed releases from Anthropic's coding assistant

⚡
**New · Caching**
**Prompt caching now lasts an hour instead of 5 minutes (if you opt in)**
You can now set `ENABLE_PROMPT_CACHING_1H=true` to make your prompts cache for a full hour instead of the default 5 minutes. This means faster responses when you're working on the same project over longer periods, since Claude won't need to re-read your entire codebase as often.

🧠
**New · Memory**
**Claude now remembers what you were doing when you come back**
Added a recap feature that gives you context when you return to a previous coding session. It's like having Claude catch you up on where you left off in your project. You can manually trigger it with `/recap` or it'll happen automatically when appropriate.

🛠️
**New · Commands**
**Claude can now find and use its own slash commands**
The AI can now discover and run built-in commands like `/init`, `/review`, and `/security-review` on its own when it thinks they'd be helpful. It's like giving Claude access to its own toolkit without you having to remember all the commands.

🔄
**Improved · Navigation**
**Better session picker shows your current project first**
When you use `/resume` to return to a previous session, it now defaults to showing sessions from your current directory first. Hit `Ctrl+A` if you want to see sessions from all your other projects too.

🔧
**Fixed · Bugs**
**Copy-paste works again · Better error messages · Memory improvements**
Fixed pasting code in the login prompt, improved error messages to distinguish between rate limits and usage limits, and reduced memory usage when working with large files. Plus several other quality-of-life fixes for a smoother coding experience.
