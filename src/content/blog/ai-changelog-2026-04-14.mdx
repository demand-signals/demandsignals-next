## Claude Code (Anthropic)

Two big updates dropped yesterday (v2.1.108 and v2.1.107):

**What changed:**
- **Smarter prompt caching:** New settings let you control how long your prompts stay cached (1 hour vs 5 minutes), which means faster responses and lower costs when you're working on the same project
- **Recap feature:** When you come back to a coding session after a break, Claude Code now gives you a quick summary of what you were working on — like having a smart assistant catch you up
- **Better session management:** The `/resume` command now defaults to showing projects from your current folder, and you get warnings before switching AI models mid-conversation
- **Bug fixes galore:** Fixed issues with pasting code, terminal display problems, and session titles getting overwritten

**Why you should care:** The prompt caching improvements will save you money if you're doing lots of back-and-forth coding with Claude. The recap feature is genuinely helpful if you jump between projects — no more "wait, what was I doing here?" moments.

This one's mainly for developers who use Claude Code's command-line tool, but if you're building software with AI assistance, these are solid improvements.
