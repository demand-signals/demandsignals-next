## Google Gemini
What's new in Google AI's conversational AI platform

⚠️
**Heads up · File Search**
**File Search tool now limits searches to 200 results**
Starting April 14, the File Search tool caps results at 200 items to keep responses focused. You can still search through thousands of files, but you'll get the most relevant 200 matches instead of being overwhelmed by every possible result.

## Claude Code
What's new from Anthropic's AI coding assistant

🧠
**New · Memory**
**Claude now remembers what you were doing when you come back**
Added a recap feature that gives you context when you return to a session. It's like having Claude catch you up on where you left off in your coding project. You can turn this on in /config or trigger it manually with /recap.

🤖
**New · Skills**
**Claude can now discover and use its own slash commands**
The model figured out how to use built-in commands like `/init`, `/review`, and `/security-review` through the Skill tool. It's basically Claude learning to help itself help you better.

🔄
**Improved · Performance**
**Lighter memory usage and smarter language loading**
File operations now use less RAM by loading syntax highlighting on demand instead of all at once. Your computer will thank you when working with large codebases.

🔧
**Fixed · Bugs**
**Better error handling and several UI fixes**
Fixed paste issues in login prompts · Clearer error messages for rate limits · Better session handling for Remote Control · Fixed character encoding problems with accents and special characters.
