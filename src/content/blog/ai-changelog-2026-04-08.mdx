**TL;DR**

OpenAI rolled out a bunch of new features for developers, but no major model updates. Meanwhile, Claude Code got a significant update with better permissions handling and bug fixes that developers will appreciate.

## OpenAI

**What changed:**
• Added new "Agents SDK" — basically pre-built AI helpers you can customize for your business
• New "ChatKit" lets you add AI chat to your website or app without building it from scratch
• Added "Computer Use" tool so AI can actually control screens and click things
• New web search and file search tools built right in
• Something called "Background mode" for long-running tasks

**Why you should care:** If you're thinking about adding AI to your business, these tools make it way easier. The Agents SDK is like having AI employees you can train for specific jobs. This one's mainly for developers — skip if that's not you.

## Claude Code (Anthropic)

**What changed:**
• New focus view (Ctrl+O) that shows just the important stuff while AI is working
• Better permission controls — won't accidentally block you from your own files anymore
• Fixed a bunch of crashes and memory leaks
• Improved file editing and session resuming
• Better handling when you're working in different directories or git branches

**Why you should care:** If your team uses Claude Code for programming, this update makes it way more reliable and less annoying to work with daily.

**The Bottom Line**

Nothing urgent today for most businesses. If you use Claude Code for development work, definitely update — the bug fixes alone are worth it. The OpenAI stuff looks cool but it's mostly for developers building custom AI tools. Regular ChatGPT users won't see any changes.
