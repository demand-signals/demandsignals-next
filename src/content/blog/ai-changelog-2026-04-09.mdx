## Claude Code
What's new from Anthropic's AI coding assistant

🛠️
**New · Setup**
**Setting up Google Vertex AI is now as easy as following a wizard**
No more digging through documentation to connect to Google Cloud. The new interactive setup wizard walks you through authentication, picking your project and region, and getting everything configured properly.

⚡
**New · Performance monitoring**
**Monitor tool lets you watch your background scripts in real-time**
If you've got scripts running in the background, you can now stream their events live to see what's happening. Perfect for keeping tabs on long-running processes without losing track.

🔑
**New · Perforce integration**
**Perforce users get proper read-only file handling**
When you set `CLAUDE_CODE_PERFORCE_MODE`, Claude Code will stop you from accidentally overwriting read-only files and suggest using `p4 edit` instead. No more "oops, I just broke the build" moments.

🛠️
**New · Security**
**Subprocess sandboxing makes your coding environment safer**
On Linux systems, you can now isolate subprocesses in their own PID namespace, plus limit how many scripts can run per session. Good news if you're working on projects where security matters.

🔧
**Fixed · Bugs**
**Bash tool security fixes · Permission bypass patches · MCP OAuth improvements**
Fixed several security issues where crafty command formatting could bypass safety checks. Also patched OAuth token refresh problems and improved permission handling across the board.
