## TL;DR

OpenAI launched a smarter version of their cheapest AI model (GPT-4o-mini) yesterday, and Claude Code (Anthropic's coding assistant) got a major update with better Google Cloud setup and tighter security features. Nothing earth-shattering, but if you use AI for basic tasks, the OpenAI update could save you money.

## OpenAI

**What changed:**
- Released GPT-4o-mini v2 — a smarter version of their cheapest AI model
- Same pricing as before ($0.15 per million input tokens, $0.60 per million output tokens)
- Better at following instructions, reasoning through problems, and handling edge cases
- Still the budget option in their lineup

**Why you should care:** If you're using GPT-4o-mini for customer emails, content writing, or simple automation tasks, this upgrade gives you better results for the same cost. Worth switching to if you haven't already.

## Claude Code (Anthropic)

**What changed:**
- Added a setup wizard that walks you through connecting to Google Cloud's AI services (no more confusing config files)
- New security features for teams using version control systems like Perforce — prevents accidental file overwrites
- Better monitoring tools for tracking what background scripts are doing
- Fixed a bunch of permission bugs that could let malicious prompts run unauthorized commands
- Improved the interface for managing multiple AI agents working together

**Why you should care:** This one's mainly for developers and technical teams. If your company uses Claude Code for software development, the security improvements are important — update soon. The Google Cloud wizard makes it easier for non-technical managers to set up AI coding assistants for their teams.

## The Bottom Line

If you're using GPT-4o-mini through OpenAI's API, the v2 upgrade is automatic and worth using. If your development team uses Claude Code, plan an update in the next week — the security fixes are important, and the new features will save your IT team setup headaches. Nothing urgent for most business owners today.
