## Claude Code
What's new from Anthropic's AI coding assistant

🧠  
**New · Team Collaboration**  
**Claude can now create onboarding guides for your team based on how you actually use it**  
Just run `/team-onboarding` and Claude will generate a teammate ramp-up guide from your local usage patterns. Perfect for getting new team members up to speed on your Claude Code workflow.

⚡  
**Improved · Enterprise Setup**  
**No more certificate headaches when your company uses proxy servers**  
Claude Code now trusts your OS certificate store by default, so it'll work with enterprise TLS proxies without any extra setup. One less IT ticket to file.

🛠️  
**Improved · Cloud Features**  
**Ultraplan and remote sessions now just work without web setup**  
When you try `/ultraplan` or other cloud features, Claude Code automatically creates a default cloud environment instead of making you go set things up on the web first. Way smoother experience.

🔧  
**Fixed · Security**  
**Patched command injection vulnerability**  
Fixed a security issue in the POSIX `which` fallback used by LSP binary detection. Also fixed several memory leaks and session recovery problems that could crash or slow things down.

🔄  
**Improved · Error Messages**  
**Better explanations when things go wrong**  
Rate limit messages now show which specific limit you hit and when it resets. Tool errors explain why a tool isn't available and what to do about it. Much less guessing what went wrong.
