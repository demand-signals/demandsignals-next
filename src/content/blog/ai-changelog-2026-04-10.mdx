## Claude Code
What's new from Anthropic's AI coding assistant

🛠️
**New · Team Tools**
**Added team onboarding command to help new teammates get up to speed**
Claude Code can now generate a personalized ramp-up guide for new teammates based on your local usage patterns. Just run `/team-onboarding` and it'll create a guide showing them the ropes.

🔑
**Improved · Enterprise Setup**
**Enterprise TLS proxies now work out of the box**
No more wrestling with certificate issues behind corporate firewalls. Claude Code now trusts your OS certificate store by default, so enterprise users can get coding right away without extra setup.

⚡
**Improved · Cloud Features**
**Remote sessions auto-create cloud environments when needed**
Features like `/ultraplan` used to require setting up a cloud environment through the web first. Now they'll automatically create one for you and get straight to work.

🤖
**Improved · Communication**
**Claude gives better explanations when tools aren't available**
Instead of cryptic "tool not available" errors, Claude now explains why a tool can't be used and suggests what to do instead. Way more helpful when you're stuck.

🔄
**Improved · Rate Limits**
**Rate limit messages now show which limit you hit and when it resets**
No more mystery countdown timers. You'll see exactly which rate limit kicked in and when you can try again, making it easier to plan your next moves.

🔧
**Fixed · Security**
**Patched command injection vulnerability in LSP detection**
This was a behind-the-scenes security fix for developers. The system that detects language server binaries had a vulnerability that's now sealed up.

🔧
**Fixed · Performance**
**Fixed memory leak in long coding sessions**
Long conversations were eating up more and more memory over time. This is now fixed, so you can code for hours without Claude Code getting sluggish.
