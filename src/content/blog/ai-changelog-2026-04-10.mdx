## Claude Code (Anthropic)

**What changed:**
- Added a `/team-onboarding` command that automatically creates guides for new teammates based on how your team actually uses Claude Code
- Made it work with enterprise TLS proxies right out of the box — no more setup headaches for corporate networks
- Cloud features like `/ultraplan` now create environments automatically instead of making you set things up on the web first
- Fixed a bunch of bugs, including a security issue with command injection and memory leaks during long coding sessions
- Improved error messages so they actually tell you what went wrong and how to fix it
- Better handling of rate limits — now shows which limit you hit and when it resets

**Why you should care:** The team onboarding feature is genuinely useful if you have developers joining your team — it learns from your actual workflow instead of giving generic advice. The enterprise proxy fix removes a common IT headache. Most other changes are quality-of-life improvements that make the tool less frustrating to use day-to-day.

**The Bottom Line**

If you use Claude Code for development work, this update is worth grabbing. The team onboarding feature alone could save hours of explaining your setup to new developers. If you're in a corporate environment, the TLS proxy fix might solve connection issues you've been dealing with.
