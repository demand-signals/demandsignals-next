## DeepSeek API
Major model upgrade brings better reasoning and speed

🧠
**New · Models**
**DeepSeek-V4 models are here with Pro and Flash variants**
DeepSeek just dropped their V4 models — V4-Pro for complex tasks and V4-Flash for speed. You can access them right now by setting your model to `deepseek-v4-pro` or `deepseek-v4-flash`. Both work with OpenAI and Anthropic API formats, so no need to change your existing code.

⚠️
**Heads up · Deprecation**  
**Legacy model names getting retired in three months**
The old `deepseek-chat` and `deepseek-reasoner` model names will stop working on July 24, 2026. For now, they still work (they point to V4-Flash), but start switching to the new V4 model names to avoid any surprises.
