## Claude Code
What's new from Anthropic's AI coding assistant

🔧
**Fixed · Bugs**
**Rate limits no longer leave you hanging**
Fixed agents getting stuck after hitting rate limits with long retry delays — now you get an immediate error instead of the tool sitting there silently waiting. Also fixed some login issues on macOS when your keychain is locked.

⚡
**Improved · Performance**
**Now thinks harder by default**
Default effort level bumped from medium to high for most users. You can still control this with the `/effort` command if you want faster responses for simple tasks.

🛠️
**New · Integrations**
**Amazon Bedrock gets an upgrade**
Added support for Amazon Bedrock powered by Mantle — just set `CLAUDE_CODE_USE_MANTLE=1` to try it out. This one's mainly for developers who need specific cloud configurations.

📋
**Improved · Interface**
**Better Slack integration**
When Claude sends messages to Slack channels, you now get a neat compact header with a clickable channel link so you know exactly where it went. Plus some fixes for text rendering and cursor tracking in terminals.
