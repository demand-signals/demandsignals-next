## Claude Code
What's new from Anthropic's AI coding assistant

🔑
**New · Cloud provider**
**Claude Code now works with Amazon Bedrock via Mantle**
If you use AWS, you can now set `CLAUDE_CODE_USE_MANTLE=1` to route your requests through Amazon Bedrock. This gives you another option beyond the standard Claude API for accessing AI models.

⚡
**Improved · Performance**
**Your coding sessions just got smarter by default**
The default effort level jumped from "medium" to "high" for most users (API-key, Bedrock/Vertex/Foundry, Team, and Enterprise). This means Claude will work harder on your code by default, though you can still adjust it with the `/effort` command.

🛠️
**New · Slack integration**
**Cleaner Slack notifications with clickable channel links**
When Claude sends messages to Slack channels, you'll now see a compact header with the channel name that you can actually click to jump straight to that channel. No more hunting around for where that message went.

🔄
**Improved · Plugin experience**
**Plugin skills get smarter naming and better session titles**
Plugin skills now use their actual names from frontmatter instead of just the folder name, so you get consistent naming no matter how you install them. Plus, plugins can now set custom session titles through hooks.

🔧
**Fixed · Bugs**
**Rate limiting errors show up immediately instead of hanging · macOS login issues now give clear error messages · Plugin hooks work properly when paths aren't set · Search inputs handle Shift+Space correctly · Links don't open duplicate tabs in certain terminals**
A bunch of annoying issues got squashed, including Claude getting stuck silently when hitting rate limits, login problems on Mac, and various plugin and interface glitches that were making the experience rougher than it should be.
