## 🚨 The Big News
DeepSeek's V4 Preview drops early, making Sunday headlines

DeepSeek V4 Pro and V4 Flash rolled out to API users Sunday, marking a significant upgrade from the V3 family. The models bring enhanced reasoning, improved coding, and better multilingual support.

**What's new:**
- Two variants: V4-Pro for complex tasks, V4-Flash for speed
- Supports both OpenAI ChatCompletions and Anthropic interfaces  
- Legacy model names (`deepseek-chat` and `deepseek-reasoner`) will be phased out in July 2026
- Same API endpoint structure, just update the model parameter

**Why this matters:** DeepSeek continues pushing competitive pressure on OpenAI and Anthropic with frontier-level performance at lower costs. The dual-interface support makes migration easier for developers already using either ecosystem.
