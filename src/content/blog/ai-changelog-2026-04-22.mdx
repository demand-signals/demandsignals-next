## Claude Code
What's new from Anthropic's AI coding assistant

🔄
**Fixed · Bugs**
**Two dozen fixes to make Claude Code more reliable**
OAuth sessions no longer die mid-session, WebFetch stops hanging on large pages, and prompt input undo actually works now. A clean-up of the little annoyances that add up over time.

🛠️
**New · External builds**
**Forked subagents now work outside Anthropic builds**
Set `CLAUDE_CODE_FORK_SUBAGENT=1` and external builds can spawn subagents just like the official version. Opens up more advanced agent workflows for teams building their own deployments.

🔧
**Improved · Plugin management**
**Plugin installs got smarter about dependencies**
Installing an already-installed plugin now grabs missing dependencies instead of giving up. The marketplace commands also auto-resolve deps from configured sources — less manual work to get plugins running.

🔄
**Improved · Session handling**
**Better model persistence and stale session recovery**
Model selections stick around even when projects pin different ones, and `/resume` offers to summarize old sessions before diving back in. Smoother workflows when switching between contexts.
