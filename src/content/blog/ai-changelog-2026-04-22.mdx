## Claude Code
What's new from Anthropic's AI coding assistant

🔄
**Improved · Configuration Management**
**Model selections now stick around after restarts, plus better plugin dependency handling**

The `/model` command now remembers your selection even when projects pin different models, and you'll see which source (project vs managed settings) controls the startup model. Plugin installation got smarter too — it'll now install missing dependencies instead of just telling you the plugin already exists.

[Source →](https://github.com/anthropics/claude-code/releases/tag/v2.1.117)

🔧
**Fixed · Bugs**  
**OAuth token refresh, HTML truncation, and context window calculations**

Fixed OAuth sessions dying with "Please run /login" when tokens expire mid-session (now refreshes automatically), WebFetch hanging on huge HTML pages, and Opus 4.7 showing wrong context percentages due to incorrect window size calculations.

[Source →](https://github.com/anthropics/claude-code/releases/tag/v2.1.117)

⚡
**Improved · Performance**
**Faster startup and native search tools on macOS/Linux**

Native builds now use embedded `bfs` and `ugrep` for file operations instead of separate tool calls, making searches much snappier. Plus faster startup when you have both local and cloud MCP servers configured.

[Source →](https://github.com/anthropics/claude-code/releases/tag/v2.1.117)
