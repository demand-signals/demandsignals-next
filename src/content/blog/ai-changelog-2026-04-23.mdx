## Claude Code
What's new from Anthropic's AI coding assistant

🔄
**Improved · Configuration Management**
**Settings that actually stick around**
Your `/config` preferences now save to `~/.claude/settings.json` instead of vanishing when you restart. Theme choices, editor modes, and verbosity settings finally persist like they should. Plus, you can set custom PR URLs if your team uses something other than GitHub.

🛠️
**New · Git Integration**
**Better PR support across platforms**
The `--from-pr` flag now works with GitLab merge requests and Bitbucket pull requests, not just GitHub. Your `owner/repo#N` links in output will also use your actual git remote instead of assuming everything lives on github.com.

🔧
**Fixed · Permissions**
**PowerShell auto-approval**
PowerShell commands can now be auto-approved in permission mode, matching how Bash commands already worked. No more manual clicks for every Windows script.

🔧
**Fixed · Paste Handling**
**Windows clipboard wonkiness**
Fixed the annoying bug where pasting content from Windows clipboards (or Xcode console) would insert extra blank lines everywhere. Multi-line pastes should now work properly in terminals using kitty keyboard protocol.

🧠
**New · Vim Visual Mode**
**Visual selection finally works**
Added vim visual mode (`v`) and visual-line mode (`V`) with proper selection, operators, and visual feedback. If you've been waiting for this, it's here.
